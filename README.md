## 遊戲概覽

| 項目 | 內容 |
|------|------|
| 玩家人數 | **2 – 8 人** |
| 建議時間 | 60 – 120 分鐘 |
| 關卡數 | **3 關**（破冰 → 深入 → 勇敢） |
| 主牌張數 | **52 張** |
| 過場牌 | **2 張**（關卡轉換行動牌） |
| 結尾提示 | **3 個**（遊戲終章） |
| 平台 | 純網頁，無需安裝，手機電腦皆可 |

## 卡牌總覽

| 類型 | 張數 | 顏色標識 | 特色 |
|------|------|----------|------|
| 🌊 **破冰**（ICE） | 12 張 | 藍色 | 輕鬆話題，降低防禦，建立安全感 |
| 🔍 **深入**（DEEP） | 18 張 | 橙色 | 真實故事，觸碰內心，深化認識 |
| ⚡ **勇敢**（BRAVE） | 13 張 | 紅色 | 需要勇氣的分享，觸及最真實的自我 |
| 🫂 **行動**（ACTION） | 9 張 | 綠色 | 互動任務，打破框架，創造當下連結 |

> **行動牌**分散嵌入在三個關卡之間（5 張）以及單獨的行動牌組（4 張），
> 另有 2 張**過場行動牌**在關卡轉換時出現。

## 如何開始

1. **打開網頁**
   前往 [https://johnbass.github.io/thinking-leverage-cards/](https://johnbass.github.io/thinking-leverage-cards/)

2. **輸入玩家名稱**
   - 預設 2 人，最多可加到 **8 人**
   - 每位玩家有專屬 emoji 圖示

3. **點擊「開始遊戲」**
   進入第一關 **破冰**，遊戲正式開始。

### 單回合流程

1. 畫面顯示**目前輪到的玩家**
2. 點擊「抽牌」按鈕，翻出一張問題牌
3. **當前玩家**大聲朗讀題目，並第一個回答
4. 其餘玩家依序補充或回應（不強制）
5. 討論結束後點擊「下一張」，換下一位玩家
6. 玩家順序**自動循環**（1→2→3→…→1→2…）

## 卡牌說明

每張牌包含三個層次的內容：

### 📖 問題
核心對話問題，設計目的是讓人**真實分享**，而非表演。

### ✝️ 聖經金句
對應題目的聖經節，提供信仰視角與屬靈深度。

### 🧠 心理學背景
說明問題背後的心理學理論（如依附理論、感恩研究、敘事心理學等），以及「**為什麼這個問題有效**」的科學解釋。

> 玩家可透過畫面底部的 **「聖經」/ 「心理學」** 標籤切換閱讀。

## 卡牌類型詳細說明

### 🌊 破冰牌（12 張）
**目的：** 建立心理安全感，讓所有人放鬆進入對話。

範例問題：
- 「如果你的人生是一部電影，你覺得現在是哪個場景？」
- 「最近有沒有一首歌讓你單曲循環？為什麼是這首？」
- 「你打包行李是早就打包好的人，還是出發前一小時才開始的人？」

---

### 🔍 深入牌（18 張）
**目的：** 進入真實的內心世界，了解彼此的故事與價值觀。

範例問題：
- 「你通常是怎麼處理壓力的？這個方法真的有效嗎？」
- 「你有沒有曾經非常確定一件事，後來卻完全改變想法？」
- 「你和金錢的關係是什麼樣的？你是容易花錢、還是很難花錢在自己身上？」

---

### ⚡ 勇敢牌（13 張）
**目的：** 觸碰最深的議題——原諒、失落、恐懼、夢想。需要彼此已建立足夠信任。

範例問題：
- 「你生命中有沒有『一個你已經原諒但還沒忘記』的傷？」
- 「你有沒有一段時期覺得人生沒有意義？那時候是什麼讓你撐過去的？」
- 「你現在最不敢讓別人知道的一面是什麼？」

---

### 🫂 行動牌（9 + 2 過場 = 11 張）
**目的：** 打破口頭對話的框架，透過具體行動創造當下的連結。

範例任務：
- 「轉向右邊的人，說出一件你第一眼見到他就注意到的正面特質」
- 「打開手機，找出你最近拍的一張照片，分享它的故事」
- 「選一位組員，用 2 分鐘只聽他說話，不打斷、不給建議」

## 設計理念

本遊戲融合兩個核心框架：

**心理學——漸進自我揭露（Progressive Self-Disclosure）**
Arthur Aron 的研究指出，當對話從低風險逐漸進展到高風險的自我揭露，親密感會顯著提升。三個關卡的設計正是依此原理，確保每個人在更深分享前已建立足夠的安全感。

**信仰——聖經的真實群體觀**
「鐵磨鐵，磨出刃來；朋友相感，也是如此。」（箴言 27:17）
真實的關係需要勇氣、謙遜與彼此的接納。每張牌的聖經金句不是裝飾，而是提供一個超越心理學的屬靈視角。

## 注意事項

1. **沒有人需要被強迫回答**
   任何題目都可以說「pass」，遊戲不是審訊，而是邀請。

2. **行動牌可調整**
   若某張行動牌在當下場合不適合，可直接換下一張。

3. **勇敢牌建議在關係建立後再進行**
   三關卡的順序設計是有意義的，不建議跳過前兩關直接抽勇敢牌。

4. **保密原則**
   建議遊戲前先約定：「在這裡說的，留在這裡。」

5. **時間彈性**
   不需要把所有牌都抽完——若某個問題引發了深度對話，就讓它發展，比張數更重要。
